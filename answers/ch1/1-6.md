# 1.6

## CompareNat1

CompareNat1 である。

例えば $1 < 4$ を 2 通りの方法で導出できる。

最後に $1 < 2$ と $2 < 4$ に `L-Trans` を適用する方法:

```text
S(Z) is less than S(S(S(S(Z)))) by L-Trans {
  S(Z) is less than S(S(Z)) by L-Succ {};
  S(S(Z)) is less than S(S(S(S(Z)))) by L-Trans {
    S(S(Z)) is less than S(S(S(Z))) by L-Succ {};
    S(S(S(Z))) is less than S(S(S(S(Z)))) by L-Succ {}
  }
}
```

最後に $1 < 3$ と $3 < 4$ に `L-Trans` を適用する方法:

```text
S(Z) is less than S(S(S(S(Z)))) by L-Trans {
  S(Z) is less than S(S(S(Z))) by L-Trans {
    S(Z) is less than S(S(Z)) by L-Succ {};
    S(S(Z)) is less than S(S(S(Z))) by L-Succ {}
  };
  S(S(S(Z))) is less than S(S(S(S(Z)))) by L-Succ {}
}
```
